# pure-node-notebook

## 项目代码要求

> 纯Node 6.10.0支持的代码，不依赖于babel

> 不依赖于任何第三方库，纯手写Node组件

> 所有语法都在Node.green下的支持列表

## 运行代码

```bash
	git clone https://github.com/slashhuang/pure-node-notebook.git
	npm install 
	npm start
```
